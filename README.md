# 🌸 Lavy's Zodiac Reading

A fun and personal Python program that greets you, reads your zodiac sign, reveals your personality based on the stars, and gives you a unique nickname!

---

## ✨ What Does It Do?

Enter your name and zodiac sign, and the program will warmly greet you, describe your personality based on your sign, and assign you one of four hand-picked nicknames — chosen randomly every time you run it. All 12 zodiac signs are included with original, personality-packed descriptions!

---

## 🔮 Signs Covered

**🔥 Fire** — Aries, Leo, Sagittarius

**🌿 Earth** — Taurus, Virgo, Capricorn

**💧 Water** — Cancer, Scorpio, Pisces

**💨 Air** — Gemini, Libra, Aquarius

---

## 📱 How to Run It on Mobile (Pydroid 3)

Pydroid 3 is a free Python IDE app for Android — perfect for running this program right on your phone!

**Step 1** — Download [Pydroid 3](https://play.google.com/store/apps/details?id=ru.iiec.pydroid3) from the Google Play Store

**Step 2** — Open the app and create a new file

**Step 3** — Copy and paste the full program code into the file

**Step 4** — Save the file as `zodiac.py`

**Step 5** — Tap the yellow play button at the bottom to run it

**Step 6** — Enter your name and zodiac sign in the terminal that appears at the bottom — and enjoy your reading! 🌟

---

## 💻 How to Run It on PC

**Step 1** — Make sure Python is installed on your computer. Download it from [python.org](https://www.python.org/) if needed

**Step 2** — Save the program as `zodiac.py`

**Step 3** — Open your terminal or command prompt

**Step 4** — Navigate to the folder where you saved the file

**Step 5** — Run the command:
```
python zodiac.py
```

## 🛠️ Requirements

- The `random` module — already built into Python, no installation needed!
